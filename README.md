MERN Stack Chat App with Socket.io
This is a real-time chat application built with the MERN stack (MongoDB, Express, React, Node.js) and Socket.io for real-time communication.

Features
1.Real-time messaging: Send and receive messages instantly using Socket.io.
2.User authentication: Register and log in with basic email/password authentication.
3.Private and group chat support: Communicate with individual users or create chat groups.
4.User presence: See who is online in real-time.
